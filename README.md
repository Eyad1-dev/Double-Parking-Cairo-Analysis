# Double-Parking in Cairo: A Research Survey

## Overview
A data analysis research project studying the severity and impact
of double-parking incidents on Cairo drivers.

## Research Question
How severely does double-parking affect Cairo drivers in terms of 
time lost, stress experienced, and frequency of incidents?

## Key Findings
- Average of 4.56 incidents per month (hypothesis confirmed)
- Average wait time of 14.5 minutes per incident (hypothesis confirmed)
- Female drivers report higher stress (7.80) vs male drivers (6.94)
- Nasr City / Heliopolis is the most affected area (~10 incidents/month)

## Files
- Report: Full research report with analysis and visualizations
- Notebook: Jupyter Notebook with all data cleaning and analysis code
- Dataset: Raw collected survey data in CSV format

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook
- Google Forms (data collection)
